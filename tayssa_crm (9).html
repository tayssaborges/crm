🔗 Ligação Google Sheets

Liga o CRM à tua folha Google para guardar os dados na nuvem e acederes de qualquer dispositivo.
Como configurar (5 passos):
1. Abre script.google.com → "Novo projecto"
2. Apaga o código existente e cola o conteúdo do ficheiro Code.gs
3. No menu superior: Executar → setup (autoriza quando pedido)
4. Clica em Implementar → Nova implementação → App Web
5. Quem tem acesso: "Qualquer pessoa" → copia o URL e cola acima

Importar / Exportar

📤 Exportar

Backup completo de todos os dados.

📥 Importar

Importa backup JSON ou CSV do Alboom.

Como importar do Alboom:
  1. No Alboom, vai a Clientes → Exportar → CSV
  2. Guarda o ficheiro no teu computador
  3. Seleciona acima em "CSV Clientes (Alboom)"
  4. Clica em "Importar CSVs"
Colunas reconhecidas automaticamente:
Nome / name / cliente  ·  Telefone / phone / telemóvel  ·  Email  ·  NIF  ·  Morada / endereço  ·  Valor  ·  Data  ·  Tipo / sessão
💡 Dica: Após importar, podes partilhar aqui o teu ficheiro CSV para verificarmos se os campos correspondem ao teu Alboom.
